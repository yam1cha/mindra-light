# Mindra Light

カスタム Electron ベースの軽量ブラウザ。


## インストール

    npm install

------------------------------------------------------------------------

## ローカルでの実行方法

    npm start

------------------------------------------------------------------------

## ローカルビルド方法（パッケージ作成）

    npm run build

`dist/` フォルダに各 OS 向けのパッケージが生成されます。

------------------------------------------------------------------------

## GitHub Actions 自動ビルド

`.github/workflows/build.yml` により次が自動生成されます：

-   Windows（.exe）
-   macOS（.app / .dmg）
-   Linux（AppImage / deb）

成果物は Artifact として保存されます。

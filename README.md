# Mindra Light

English version: README_en.md

**Mindra Light** は、APIキーを必要とせず、  
無料で使えるローカル LLM を活用した AI ブラウザです。

外部の AI サービスに依存せず、  
個人の環境で AI とブラウジングを組み合わせることを目的としています。

スプリットビュー、2段階プロファイル、垂直タブなど、  
既存のブラウザとは異なる独自の UI・操作性を取り入れています。

---

## プロジェクトについて

Mindra Light は、  
「個人が AI をどのように日常の作業に組み込むか」をテーマにした  
個人開発の AI ブラウザです。

APIキーや課金を前提とせず、  
ローカル環境で完結する AI 活用を重視しています。

機能や UI は、実際の利用を通じて継続的に改善されています。

---

## 特徴

- APIキー不要で利用できるローカル LLM 連携
- 無料で使える AI 環境を前提とした設計
- スプリットビューによる複数ページ同時表示
- 2段階プロファイルによる用途別ブラウジング
- 垂直タブによる高い視認性と操作性
- 既存のブラウザとは異なる独自の UI 構成

## インストール（利用者向け）

1. GitHub の Releases ページを開く  
2. お使いの OS に対応したファイルをダウンロード  
3. 展開またはインストールして起動  

※ Linux では AppImage / deb を利用できます。

---

## macOS 利用時の注意

macOS では、未署名のアプリに対して  
初回起動時に警告が表示される場合があります。

その場合は、以下の手順で起動できます。

1. アプリを右クリック（または Control + クリック）
2. 「開く」を選択
3. 表示される確認ダイアログで再度「開く」を選択

または、

- 「システム設定」→「プライバシーとセキュリティ」
- 下部に表示される「このアプリの実行を許可」を選択

これは macOS のセキュリティ仕様によるもので、  
アプリ自体の問題ではありません。

## ローカル LLM（Ollama）について

Mindra Light は **Ollama を利用したローカル LLM** と連携することで、  
APIキー不要の AI 機能を提供します。

### Ollama のインストール

#### Windows / macOS
https://ollama.com/download

#### Linux（Ubuntu 例）

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

---

### 推奨モデル

```bash
ollama pull qwen2.5:7b-instruct
```

---

### 動作確認

```bash
ollama run qwen2.5:7b-instruct
```
